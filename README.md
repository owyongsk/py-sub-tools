# py-sub-tools
A collection of useful Python subtitle related tools

## Requirements

Python2 and Python3

## Usage

1. `git clone`
1. `git submodule init`
1. `git submodule update`
1. I like to `alias` on my `./bash_profile` or `./zprofile` to the appropriate commands. Eg: add `alias osd=~/path/to/py-sub-tools/OpenSubtitlesDownload/OpenSubtitlesDownload.py` to your `./bash_profile`
1. Add remaining aliases for the projects, they have `--help` options
